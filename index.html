Professional Experience and Projects

Explore the work and achievements of Ashwin Chembu

Tallyrus

Founder and Lead Developer of Tallyrus, a platform utilizing LangChain, Pinecone, and OpenAI for automated essay grading via a Retrieval-Augmented Generation (RAG) pipeline. Enhanced educational efficiency across multiple schools by building a scalable backend with Flask and MongoDB and implementing a dynamic frontend using React and Node.js.

Figma Chat Widget

Developed an interactive chat widget integrated into a FigJam dashboard to enhance real-time feedback in educational settings. The widget features a robust backend using Express, MongoDB, and admin controls, improving the collaborative experience for nearly 4,000 users.

Echo

Collaborated with Professor Mohammad Sadoghi at UC Davis on Echo, a digital verification platform leveraging MXFace (facial detection ML API) and secure image storage using AWS S3. Won the Solana prize at HackMIT and further integrated blockchain technology via ResilientDB.

Business Technology Solutions Associate

At ZS Associates, specialized in developing Python endpoints with AWS S3 and the Smartsheet API for automated document management, improving data retrieval efficiency for clients like Amgen. Also enhanced and tested RAG models and MosaicML workflows.

Tech4Good @ UCSC

Led the development of key widgets (carousel, chat, polling) using JavaScript, React, Node.js, and MongoDB. Improved real-time collaboration for 4,000 users and mentored interns in delivering high-quality development, ensuring robust backend performance.

File System Project

Developed ECS150-FS, a simple file system based on a FAT structure. Implemented virtual disk management, file operations, and metadata handling, enhancing understanding of FAT-based file systems and C programming.
