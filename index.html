Private Filedrop

Public repository · Deployed on Render

A password-protected file-exchange service for moving files between computers and AI sessions without exposing operational data or credentials.

Problem

Cross-device and AI-session file handoffs were slow and easy to lose.

Approach

Unify browser uploads, APIs, MCP tools, versioning, and messages.

Engineering

Node.js, Express, OpenAPI, S3-compatible storage, and signed webhooks.

Outcome

A durable bridge with explicit boundaries between automation and human control.

Node.jsExpressS3OpenAPIMCPHMAC
Architecture & security details

The service exposes REST, OpenAPI, and MCP interfaces over local or S3-compatible storage, with shared channels for cross-session coordination.

  • Implemented categories, tags, project paths, version history, unguessable downloads, and storage migration tooling.
  • Added API-key access, timing-safe credential checks, upload validation, and HMAC-SHA256 signed webhooks.
  • Documented browser, API, AI-client, storage-migration, and deployment workflows without publishing secrets or user data.
