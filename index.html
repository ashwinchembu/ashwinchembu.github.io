03 · Systems programming

ECS150-FS

A File Allocation Table file system built in C over a virtual disk. It manages persistent metadata, free-block allocation, descriptors, directory state, and complete read/write behavior across mount cycles.

CFATVirtual diskSystems debugging
Implementation details

Implemented mount and unmount, file and directory creation, deletion, descriptor handling, seeking, and persistent reads and writes against a disk abstraction.

  • Kept FAT entries, directory metadata, and file data synchronized through allocation and deletion paths.
  • Tested concurrent access, large files, overwrite behavior, filesystem limits, and error handling.
  • Debugged edge cases in block chaining, persistence, and memory ownership at the systems boundary.
